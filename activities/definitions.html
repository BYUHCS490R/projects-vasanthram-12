Internet

The Internet is a network of networks that links computers globally. These computers can then share information with each other.

Web

The Web is basically the set of all sites that one can access through a browser like Google or YouTube.

How are the Internet and Web different?

The Internet is the large network connecting everything while the web is only one of its parts which demonstrates websites.

ASCII and Unicode

ASCII was a method for representing letters and characters by numbers. Unicode is a much better system that represents characters of almost all the languages of the world.

What is UTF-8

UTF-8 is a popular method of encoding characters in Unicode. It is able to keep characters of the English language very compact and at the same time be capable of other characters.

URL

A URL is the location of a web page that is entered in a browser, for example, google.com.

Domain Name

Domain names represent the simple words for the websites e.g. facebook.com that make it easier for us to remember instead of numbers.

HTTP

HTTP is the protocol that defines how you browser can request pages from a server.

HTTPS

HTTPS is basically the same as HTTP but with added security features. It makes sure the connection is encrypted and keeps the user data safe.

IP Address

An IP address is a unique identifier allotted to each device so that it can be located on the Internet.

DNS

DNS is a service which translates a website's name (amazon.com) into the corresponding IP address that a computer can understand.

W3C Recommendations

W3C Recommendations are standards designed to help websites be compatible with different browsers and devices.

XML, XHTML, HTML, HTML5

XML is a format designed for data storage and exchange. XHTML is a more strict version of HTML. HTML is a language that is used to create the structure of the web pages. HTML5 is the latest version of HTML that has got more functionalities such as video and audio.
