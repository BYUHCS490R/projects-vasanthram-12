Form Basics

Forms

Gender:

  • The action attribute in a form tag defines where the form data will go after a submission. It is essentially the browser command of which page or server is going to work with the user data.
  • The method attribute is about the data transmission from the form. It typically chooses between GET and POST for server communication.
  • Contrast POST with GET where the latter puts data in the URL, hence visible, and the former stores it in the request body, thus being a more secure way of handling sensitive data.
